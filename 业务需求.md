命题1 面向课堂观察报告的维度解读Skill 的实现 与 评测优化闭环

* 需求背景：
  * 当前课堂观察报告页面中的各项报告解读，主要采用“统一 System Prompt + 当前维度数据 + LLM”的方式直接生成。由于模型输入通常只包含当前维度的数据，解读视角较为单一，难以结合其他相关维度进行交叉分析，容易出现结论片面、依据不足以及不同维度解读方式同质化等问题。
* 需求内容：
  * 基于Skill 设计特定维度的角度方式，在分析当前维度时，能够按需引用其他相关维度的数据进行补充、验证或归因，形成更加全面、有依据的多维度分析结果。
  * 各维度 Skill 的输出需接入命题二所建设的 Agent 测评平台，通过统一测评集进行自动化评估，并根据测评结果定位问题、调整解读策略，实现“生成—测评—优化—再测评”的自动化迭代闭环。

挖掘不同数据之间的关系，如学生学习数据是否能为学生互动数据的ai分析提供帮助，进一步产出更完整、更有针对性的分析报告
分析希沃课堂智能反馈系统学生学习及教师教学的多维度数据，挖掘各分块数据之间的关联，提出一版业务skill生成及数据集评测依据
