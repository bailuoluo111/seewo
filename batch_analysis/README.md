# 希沃课堂数据批量提取工具

## 功能说明

从Excel文件中批量提取希沃课堂的所有分析数据，汇总成一个Excel文件。

**输入**：包含课程链接的Excel文件（`【企微登录使用】对外演示的课堂反馈报告.xlsx`）

**输出**：一行一个课程的汇总Excel，包含50+个数据字段

## 数据字段

### 基本信息

- course_id, course_name, teacher_name, school, stage, subject, classroom
- class_start, class_end

### Skill 1: 学生互动

- 平均抬头率(%), 平均举手率(%), 平均参与度(%)

### Skill 2: 学习行为

- 学习行为_被动学习占比(%), 学习行为_讨论占比(%), 学习行为_实践占比(%)
- 学习行为_总时长(秒), 估算知识留存率(%)

### Skill 3: SOLO分类

- SOLO_总回答数
- SOLO_前结构/单点结构/多点结构/关联结构/抽象拓展：数量 + 占比(%)

### Skill 4: 应答时间

- 应答_总回答数
- 应答_≤5秒/5-15秒/>15秒：数量 + 占比(%)

### Skill 5: 讲授分析

- 讲授_字数, 讲授_时长(秒), 讲授_平均语速(字/秒)

### Skill 6: 课堂流程重构

- 课前→课中_链接等级, 课中→课后_链接等级

### Skill 7: 提问有效性

- 布鲁姆_总问题数
- 布鲁姆_记忆/理解/应用/分析/评价/创造：数量 + 占比(%)
- 布鲁姆_高阶思维占比(%)
- 理答_总次数
- 理答_针对性肯定/简单肯定/启发鼓励/追问/引导：数量 + 占比(%)
- 理答_高质量理答占比(%)
- 提问有效性得分

## 使用方法

### 1. 基本用法（使用默认配置）

```bash
cd /Users/zwj/Projects/seewo/batch_analysis
python3 batch_extract.py
```

这会：

- 读取父目录下的 `【企微登录使用】对外演示的课堂反馈报告.xlsx`
- 自动生成输出文件：`课程数据汇总_YYYYMMDD_HHMMSS.xlsx`
- 每个课程之间间隔0.5秒

### 2. 指定输出文件

```bash
python3 batch_extract.py --output 我的分析结果.xlsx
```

### 3. 指定Excel源文件

```bash
python3 batch_extract.py --excel /path/to/your/file.xlsx
```

### 4. 调整请求间隔（避免速率限制）

```bash
python3 batch_extract.py --delay 1.0  # 每个课程间隔1秒
```

### 5. 查看帮助

```bash
python3 batch_extract.py --help
```

## 输出说明

### 主数据表（sheet: 课程数据汇总）

每行一个课程，包含所有数据字段

### 失败记录表（sheet: 提取失败）

如果有课程提取失败，会单独记录在这个sheet中，包含：

- course_id
- error（失败原因）

## 注意事项

1. **Token配置**：确保 `SEEWO_CONFIG.md` 中配置了有效的 token 和 username
2. **网络连接**：需要能够访问 `https://edulyse.seewo.com` API
3. **处理时间**：36个课程大约需要20-30秒（取决于网络速度和间隔设置）
4. **错误处理**：单个课程失败不会中断整个流程，会记录到失败列表中

## 故障排查

### Token失效

如果遇到 401/403 错误，请更新 `SEEWO_CONFIG.md` 中的 token

### 某个课程一直失败

可能是该课程数据不完整或API返回格式异常，查看错误信息

### 速率限制

如果遇到429错误，增加 `--delay` 参数值
